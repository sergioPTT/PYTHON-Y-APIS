Requisitos
En este ejercicio, buscamos que des tus primeros pasos como desarrollador. Crearás una API simple utilizando FastAPI en Python para gestionar una lista de tareas pendientes (retador, ¿cierto?). Pues bien, la API permitirá realizar operaciones básicas como las siguientes:

obtener todas las tareas;
obtener una tarea por su ID;
crear una nueva tarea;
marcar una tarea como completada;
y eliminar una tarea.
Instrucciones
La lista de tareas pendientes puede simularse como una lista de diccionarios que representan las tareas, en la que cada tarea cuente con id, descripción y un boolean que indique si está completada. Por ejemplo:

# Lista de tareas simulada
tasks = [
    {"id": 1, "description": "Hacer la compra", "completed": False},
    {"id": 2, "description": "Pagar las facturas", "completed": False},
]
Formato: debes realizar el ejercicio en un único archivo main.py.

Mediante FastAPI, deberás preparar los siguientes métodos:

GET /tasks/: este endpoint permitirá obtener todas las tareas pendientes y devolverá tasks.
GET /tasks/{task_id}: este endpoint permitirá obtener una tarea específica por su ID, o devolverá un error 404 si la tarea no existe.
POST /tasks/: este endpoint permitirá crear una nueva tarea, pasándole una descripción para la nueva tarea, y la agregará a la lista de tareas pendientes. Devolverá la nueva tarea creada.
PUT /tasks/{task_id}: este endpoint permitirá marcar una tarea ya existente como completada. Devolverá los detalles de la tarea actualizada o devolverá un error 404 si la tarea no existe.
DELETE /tasks/{task_id}: este endpoint permitirá eliminar una tarea existente. Devolverá los detalles de la tarea eliminada o un error 404 si la tarea no existe.
Puedes lanzar la aplicación desde una terminal, ejecutando uvicorn main:app --reload(ignora la ventana emergente de 'Open in Browser').
